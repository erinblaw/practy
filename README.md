## Supported Languages

## Installation and Deployment

Prequisites:

* Python 3
* Python Module: virtualenv

```bash
python -m venv venv

. venv/bin/activate

pip install --upgrade pip

pip install -r requirements

mkdocs serve
```

Open `http://127.0.0.1:8000/` in a browser
