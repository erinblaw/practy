# Running JavaScript

## MacOS

### Install Homebrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

brew update

brew doctor
```

### Install Node

```bash
brew install node
```

### Create Working Directory

```bash
mkdir practice/javascript

cd practice/javascript
```

### Run `test.js`

```bash
echo "print('Test Successful!')" > test.js

node test.js
```
